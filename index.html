6. Revise Functions and Scope

Watch Video 1 Watch Video 2
Practice Questions:
  • A sum-product number is a number in which the sum of the digits equals the product of the digits. Write a program in C to accept a number and check whether it’s a sum product number. Example: 123 (1+2+3 = 6 and 1*2*3 = 6).
  • Accept a number and check whether it is a Harshad Number. Example: 21 is a Harshad Number (2+1 = 3, and 21 is divisible by 3).
  • Accept a number and check whether it is a Duck Number. Example: 5076 is a Duck Number; 6742 is not.
  • A number is said to be a Disarium number when the sum of its digits raised to the power of their respective positions is equal to the number itself. Example: 135 (1^1 + 3^2 + 5^3 = 135).
  • A Prime Number is said to be a “Twisted Prime” if the new number obtained after reversing the digits is also a prime number. Write a program to accept a number and check whether the number is “Twisted Prime” or not. Example: 167.
  • Accept a number and print the number after removing all the zeros from the number. Example: Input: 54002007; Output: 5427.

8. Learn Patterns

Watch Video 1 Watch Video 2 Watch Video 3
Practice Questions:
  • Write a Menu-Driven Program to accept a number from user and check whether it is a Prime Number or an Automorphic Number according to the user’s choice.
    • Use the following functions:
    • int chkAutomorphic(int n) -> Checks whether the number is automorphic or not.
    • int chkPrime(int n) -> Checks whether the number is prime or not.
  • Write a menu-driven Program:
    • To find and display all the factors of a number input by the user (including 1 and excluding the number itself). Example: Input: n=15; Output: 1, 3, 5.
    • To find and display the factorial of a number. Use a separate function to calculate the factorial of the number.
  • You are asked to make a program that will accept numbers from the user and give its factorial. There is no limitation on how many numbers the user will input, it will display the factorial of the number. The process will stop, once the user enters 0 or any negative number. For example, the user enters 5, the program will display 5! (5 Factorial) and then ask for next input. Again user enters 7, then the program will display 7! and ask for next input. Now the user enters -5, so the program will terminate now. Also use a function to calculate the factorial
  • Write a program to print only the prime and odd Fibonacci numbers.
